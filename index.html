당근마켓 상세페이지입니다

당근마켓 WEB을 클론코딩하였습니다.
당근마켓에서 주로 사용하는 부분이 >상품업로드 및 지도, 상품등록 유저끼리 소통이 가능하게하는 웹 채팅을 구현하였습니다.

SpringBoot3 / Java / JavaScript / OracleDB / Stomp 와 SockJS로 채팅을 만들고
드래그앤드롭을 만들어 유저가 간편하게 사진을 게시할 수 있게 구현하였습니다.
또한 MyBatis동적쿼리를 사용하여 검색기능도 추가하였습니다.
마이페이지에서 회원 프로필 사진 변경 업로드 및 회원 수정 탈퇴 기능을 추가하였고
회원의 행동(게시글조회, 좋아요 등등) 을 DB에 저장하여 관리자 페이지에서 확인이 가능하도록 구현하였습니다.
